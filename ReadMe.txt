//////////////////////////
// HOW TO COMPILE
//

The engine source requires the DirectX SDK (June 2010) version.
Newer versions are untested.

http://www.microsoft.com/en-us/download/details.aspx?id=6812

Install to the default directory.


To compile the custom model exporter, the source requires the 
3ds Max 2012 SDK.  Newer versions are untested.

http://usa.autodesk.com/adsk/servlet/item?siteID=123112&id=16707768
